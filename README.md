# 아들을 위한 퀴즈 앱 (Quiz For Son) 🎮

정우를 위해 만든 특별한 퀴즈 모험 앱입니다! 
다양한 카테고리의 퀴즈를 풀며 재미있게 학습할 수 있습니다.

## 🚀 바로 가기 (Live Demo)
이 앱은 GitHub Pages를 통해 실시간으로 확인하실 수 있습니다:
**[퀴즈 앱 바로가기](https://SeungMin-Park-psm1757.github.io/Quiz_for_son/)**

## ✨ 주요 기능
- **다양한 카테고리**: 동물, 과일 등 아이들이 좋아하는 주제들로 구성.
- **음성 지원 (TTS)**: 문제를 읽어주는 기능으로 정우가 혼자서도 즐길 수 있어요.
- **예쁜 디자인**: 아이들의 눈높이에 맞춘 밝고 선명한 디자인.
- **결과 화면**: 퀴즈를 다 풀고 나면 점수와 함께 칭찬을 해줘요!

## 🛠 사용된 기술
- **React Native / Expo**: 크로스 플랫폼 앱 개발 프레임워크.
- **Firebase**: 데이터 저장을 위한 백엔드.
- **Expo Speech**: 텍스트를 음성으로 변환.

## 📦 설치 및 실행 방법

### 로컬 실행
1. 저장소를 클론합니다:
   ```bash
   git clone https://github.com/SeungMin-Park-psm1757/Quiz_for_son.git
   ```
2. 종속성을 설치합니다:
   ```bash
   npm install
   ```
3. ExpoGo 또는 웹에서 실행합니다:
   ```bash
   npx expo start
   ```
